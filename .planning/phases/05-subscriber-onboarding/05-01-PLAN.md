**Tier-specific highlight steps** (3 per tier, from research):
- Explorer: Startup Ecosystem (/dashboard/network), Investment Landscape (/dashboard/landscape-intro), Geography Map (/dashboard/map)
- Analyst: Magic Quadrant (/dashboard/quadrant), Geography Map (/dashboard/map), Category Treemap (/dashboard/treemap)
- Strategist: Company Compare (/dashboard/compare), Investor Network (/dashboard/investor-network), Sankey Flow (/dashboard/sankey)
- Admin: same as Strategist (admins skip onboarding anyway, but define for safety)

**Each step object:**
```typescript
{ title: string, description: string, href: string, icon: LucideIcon }
```
Use icons from lucide-react: GitBranch (Quadrant), Map (Geography), Network (Ecosystem), BarChart3 (Treemap), ArrowRightLeft (Compare), Users (Investor Network), Workflow (Sankey), Compass (Landscape).

**Component structure:**
1. `useState<number>(0)` for current step index
2. `useState<boolean>(true)` for dialog open state
3. Select steps array based on `accessTier` prop
4. Use existing `Dialog` component with `open` and `onOpenChange` props
5. `DialogContent` with `showCloseButton={false}` (use explicit Skip button instead)
6. `DialogHeader` with `DialogTitle` showing step title and step indicator (e.g., "Step 1 of 3")
7. `DialogDescription` with the step description text
8. Icon rendered in a colored circle above the title (use `bg-primary/10 text-primary` pattern matching ScenarioPicker cards)
9. `DialogFooter` with:
   - "Skip" button (variant="ghost") -- calls handleComplete
   - "Back" button (variant="outline") -- only shown when step > 0
   - "Next" / "Start Exploring" button (default variant) -- advances step or calls handleComplete on last step
10. Step indicator dots below content: 3 small circles, active one uses `bg-primary`, others `bg-muted`

**handleComplete function:**
```typescript
async function handleComplete() {
  try {
    await fetch('/api/profile/onboarding', { method: 'POST' })
  } catch {
    // Best-effort -- don't block user if API fails
  }
  setOpen(false)
  onComplete()
}
```

**Styling:** Professional B2B tone. No animations beyond the Dialog's built-in fade. No confetti, no gamification. Clean, minimal, informative. Max width `sm:max-w-md` on DialogContent.
